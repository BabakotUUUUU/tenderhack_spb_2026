# BPMN-схема: Поиск цен в открытых источниках

Диаграмма: `docs/bpmn/price-search-process.mmd`

## Как экспортировать

1. Открыть [mermaid.live](https://mermaid.live/)
2. Вставить содержимое `.mmd` файла
3. Actions → Export PNG/SVG → приложить к презентации

## Процесс:

1. Пользователь вводит товар и регион.
2. Frontend отправляет запрос в FastAPI.
3. Backend нормализует запрос.
4. NLP исправляет опечатки и расширяет запрос синонимами.
5. Backend параллельно запускает четыре источника через Playwright: Wildberries, Ozon, Яндекс Маркет. Рунет — собственный crawler + SQLite FTS5.
6. Каждый источник выполняется безопасно: ошибка, timeout или пустой ответ превращаются в статус источника, а не в общий краш.
7. Backend очищает товары, нормализует цены и ссылки.
8. Backend удаляет дубли.
9. Backend ранжирует товары по совпадению запроса и полноте карточки.
10. Backend группирует выдачу по источникам и возвращает `meta`.
11. Frontend показывает группированные результаты, статусы источников и обработку запроса.
12. Пользователь переходит к первоисточнику.

Экспорт PNG/PDF: открыть `.mmd` в Mermaid Live Editor и экспортировать диаграмму для презентации.
